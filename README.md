# Healthcare Data Analysis

## 📌 Project Overview
This project analyzes healthcare data to identify disease-wise treatment costs and trends. The goal is to extract meaningful insights and support data-driven decision making.

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib

## 📊 Key Features
- Data cleaning and preprocessing
- Disease-wise cost analysis
- Data visualization using graphs
- Identifying trends and patterns

## 📈 Insights
- Certain diseases have higher treatment costs compared to others
- Most patients fall within specific age groups
- Visualizations help understand healthcare trends effectively

## 🚀 Conclusion
This project demonstrates basic data analysis skills and the ability to work with real-world datasets using Python.
